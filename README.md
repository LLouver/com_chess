###通讯指令
```java
/host id
    //主持一局游戏，id为游戏唯一识别编号
//只有id匹配才执行下面的指令
/settime id 200 30 
    //设置局时、步时分别为200s，30s
/ready id b 1
    //side:1白0黑 b:1-准备/0-取消准备
/begin id 
    //比赛开始
/end id e info
    //游戏结束，1-白胜 0-黑胜 e-平 info:(0：将杀 1：认输 2：超时)

    

```
